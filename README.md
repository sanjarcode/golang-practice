## Learning Resources

1. **Go by Example** ([gobyexample.com](http://gobyexample.com/)) — 2–3 hour syntax orientation, self-paced, no handholding. Annotated examples.

2. **Exercism Go Track** ([exercism.org/tracks/go](http://exercism.org/tracks/go)) — 164 exercises with idiomatic feedback and mentoring. Core practice hub.

3. **100 Go Mistakes and How to Avoid Them** ([100go.co](http://100go.co/)) — Read in parallel with Exercism, dip in by topic as you encounter each concept.

4. **Go Bootcamp: 1000+ Exercises and Projects** (Udemy — Inanc Gumus) — For deep dives into Go internals (slices, maps, memory layout) and small projects.

5. **YouTube Playlist — videos 69–84 only** (Code & Learn) — Concurrency deep dive: goroutines, sync primitives, patterns, pipelines.
